List databases owned by a wallet.

An owner can be specified with the --owner flag. If no owner is specified, the locally configured wallet is used.

### Usage

```
kwil-cli database list [flags]
```

### Example

```
# List databases owned by a wallet

$ kwil-cli database list --owner 0x9228624C3185FCBcf24c1c9dB76D8Bef5f5DAd64
Databases belonging to '0x9228624C3185FCBcf24c1c9dB76D8Bef5f5DAd64':
 - mydb (dbid: x650ec2637346d022d9af95bd12cbac0733af785b92449e4d3b866987)
 - mydb2 (dbid: xd9e21038623fd4da9067ada21a0f272934ec6b08e47daef004f143b4)
```

### Options

```
  -h, --help           help for list
  -o, --owner string   The owner of the database
```
