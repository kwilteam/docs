Oracles allow nodes to listen to external systems and automatically create and vote on resolutions. This is primarily used for syncing data from
external event-driven systems, such as Ethereum. It can also be used to poll external APIs for data, as long as the exact data can eventually be agreed upon
by the requisite threshold for
the resolution to pass.
