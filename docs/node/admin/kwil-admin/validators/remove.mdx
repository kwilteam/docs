Command `remove` votes to remove a validator from the validator set. If enough validators vote to remove the validator, the validator will be removed from the validator set.

### Usage

```
kwil-admin validators remove <validator> [flags]
```

### Example

```
$ kwil-admin validators remove e16141e4def3a7f2dfc5bbf40d50619b4d7bc9c9f670fcad98327b0d3d7b97b6 --key-file "~/.kwild/private_key"
TxHash: a001d7ccf73b05d6aa0d749d25cec60fef4de606533ac21f48043c7cc65dfe1b
```

### Options

```
  -h, --help                 help for remove
  -k, --key-file string      file containing the private key of the local validator that will sign the transaction (mutually exclusive with --private-key))
  -p, --private-key string   hex string of the private key of the local validator that will sign the transaction (mutually exclusive with --key-file)
```
