## kwil-admin validators remove

Command `remove` votes to remove a validator from the validator set.

### Synopsis

Command `remove` votes to remove a validator from the validator set. If enough validators vote to remove the validator, the validator will be removed from the validator set.

```
kwil-admin validators remove <validator> [flags]
```

### Examples

```
# Remove a validator from the validator set, by hex public key
kwil-admin validators remove e16141e4def3a7f2dfc5bbf40d50619b4d7bc9c9f670fcad98327b0d3d7b97b6
```
