Print a list of the node's peers, with their public information.

### Usage

```
kwil-admin node peers [flags]
```

### Example

```
# Print a list of the node's peers
kwil-admin node peers --rpcserver localhost:50151 --authrpc-cert "~/.kwild/rpc.cert"
```

### Options

```
      --authrpc-cert string   kwild's TLS certificate
  -h, --help                  help for peers
  -s, --rpcserver string      admin RPC server address (default: 127.0.0.1:50151) (default "127.0.0.1:50151")
      --tlscert string        kwil-admin's TLS certificate file for server to authenticate us [default: auth.cert] (default "auth.cert")
      --tlskey string         kwil-admin's TLS key file to establish a mTLS (authenticated) connection [default: auth.key] (default "auth.key")
```
