## kwild validators join-status

Get the status of a pending validator join request.

### Synopsis

The `query` command gets the status of a pending validator join request.

```
kwild validators join-status <joiner> [flags]
```

### Examples

```
# Query the status of a pending validator join request, by providing the validator info in format <hexPubkey#pubkeytype>
kwild validators join-status 6ecaca8e9394c939a858c2c7b47acb1db26a96d7ab38bd702fa3820c5034e9d0#0
```
