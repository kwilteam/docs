`kwild` command starts the blockchain and the database services of the Kwil node. 

### Quickstart a Kwil node:

Run the below command to quickstart a Kwil node without specifying any configuration.
```bash
kwild --autogen
```
This command will generate the missing configuration in the location `~/.kwild` and starts the node with the generated configuration. 

### Run node with specific configuration

All the configuration required for running a Kwil node has to be located under the root directory. The default root directory used is `~/.kwild` and can be overwritten using the `--root_dir` command line flag.

If the root directory is initialized with the required configuration, then the node can be started by running the below command:
```bash
kwild --root_dir "path/to/root/dir"
```

### Override configuration using command line flags

The `kwild` command also supports overriding the configuration using the command line flags.  Use `--help or -h` for detailed information on all the command line flags supported by the `kwild` CLI. For more information, refer to the [configuration guide](./configuration/configuration.mdx#config-override)
