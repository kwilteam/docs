Kwild server commands provides a way to interact and operate kwild server.
```
kwild server --help

Commands to run Kwild Server

Usage:
  kwild server [command]

Available Commands:
  start       kwil grpc server

Flags:
  -h, --help   help for server

Global Flags:
      --config string   kwild config file (.toml)

Use "kwild server [command] --help" for more information about a command.
```



To start the Kwild server:
`kwild server start --config "config-file-path"`

All the kwild application and chain configuration defined in the config.toml can be overwritten using flags while starting the server.
