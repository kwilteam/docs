The kwild command line interface (CLI) are used to configure and manage kwild nodes.

### Basic Usage

``` bash
kwild --help
kwild allows you to configure Kwild services

Usage:
  kwild [command]

Available Commands:
  help        Help about any command
  server      Kwild Server commands
  utils       Utility commands to generate and view files required for kwil network
  validator   manage validators

Flags:
  -h, --help          help for kwild
      --config string   kwild config file (.toml)

Use "kwild [command] --help" for more information about a command.
```

#### Main Commands

- `server`: Contains subcommands specific to running the kwild node.

- `validator`: Contains a suite of subcommands that are useful for validator management. These commands facilitate operations such as joining or leaving a network as a validator, granting approval to a validator join, and checking a validator's status.

- `utils`: Contains utility commands that help in generating and viewing files necessary for setting up the Kwild network.

#### Global flags

- `--config (string)`: Specifies the kwild config.toml file.
