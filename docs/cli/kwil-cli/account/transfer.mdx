## kwil-cli account transfer

Transfer value to an account

### Synopsis

Transfers value to an account.

```
kwil-cli account transfer <recipient> <amount> [flags]
```

### Options

```
  -h, --help        help for transfer
  -N, --nonce int   nonce override (-1 means request from server) (default -1)
      --sync        synchronous broadcast (wait for it to be included in a block)
```
