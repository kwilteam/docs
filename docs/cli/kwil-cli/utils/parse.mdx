## kwil-cli utils parse

Parses SQL and returns the AST.

### Synopsis

Parses SQL and returns the AST.

It can either be given a file or a string on the command line to parse.
It can be given an out file where it will write the AST to a file.
If no out file is given, it will simply print whether it was successful or not.

```
kwil-cli utils parse [flags]
```

### Options

```
  -h, --help               help for parse
  -i, --in string          A file that SQL should be read from.
  -o, --out string         Output file to write the AST to.
  -r, --remove-positions   Remove positions from the AST.
```
